11.18.2017

Some distinctive features that might distinguish 'informational' searches from 'navigational' searches are:

  • Whether a search term contains 'question' words (like what, when, how) - which might suggest an informational search as opposed to a navigational search.
  • Whether a term ends in a question mark - which again might suggest an informational search.
  • Whether a term contains the word 'page', or perhaps the name of a specific website (like 'clinton wikipedia') - which might suggest a navigational search.
